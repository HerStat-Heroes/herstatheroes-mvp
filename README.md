# HerstatHeroes MVP

This project is the backend server for the HerstatHeroes application, developed during a hackathon. The application provides detailed analytics and visualizations for baseball players' performance. The backend is built using Flask and provides various endpoints to fetch and visualize data related to baseball events.

## Requirements

- Python 3.7+
- Flask
- numpy
- pandas
- json

## Setup

1. **Clone the repository:**
   ```sh
   git clone https://github.com/yourusername/herstatheroes-backend.git
   cd herstatheroes-backend
   ```

2. **Create a virtual environment and activate it:**
   ```sh
   python3 -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install dependencies:**
   ```sh
   pip install -r requirements.txt
   ```

4. **Run the Flask application:**
   ```sh
   flask run
   ```

## Project Components

### Directories and Files

- `dataset/`: Contains the dataset files in JSONL format.
- `static/`: Static files (CSS, images) used by the templates.
- `templates/`: HTML templates used for rendering web pages.
- `videos/`: Directory to store video files generated by the backend.
- `.env`: Environment variable configuration file.
- `app.py`: Main Flask application script.
- `data_model.py`: Script containing data analysis and model functions.
- `video.py`: Script for video generation.
- `womenintech_squared_up_rate_predict.ipynb`: Jupyter notebook for data analysis.

## Endpoints

### `/`
- **Method:** GET
- **Description:** Render the homepage.

### `/data/<data_id>`
- **Method:** GET
- **Description:** Fetch data by ID.

### `/pitch_events`
- **Method:** GET
- **Description:** Fetch all pitch events.

### `/hit_events`
- **Method:** GET
- **Description:** Fetch all hit events.

### `/teams`
- **Method:** GET
- **Description:** Fetch all team IDs.

### `/hitters`
- **Method:** GET
- **Description:** Fetch all hitter IDs.

### `/team_profile/<team_id>`
- **Method:** GET
- **Description:** Render the team profile page.

### `/hitter_profile/<hitter_id>`
- **Method:** GET
- **Description:** Render the hitter profile page.

### `/visualization/<data_id>`
- **Method:** GET
- **Description:** Render the visualization page for the given data ID.

### `/video/<data_id>`
- **Method:** POST
- **Description:** Generate video for the given data ID.

### `/video/<data_id>/<view_id>`
- **Method:** GET
- **Description:** Fetch a specific view of the generated video.

### `/delete_videos/<data_id>`
- **Method:** DELETE
- **Description:** Delete all videos for the given data ID.

### `/extract_info/<data_id>`
- **Method:** GET
- **Description:** Extract specific information from the given data ID.

### `/data_by_team/<team_id>`
- **Method:** GET
- **Description:** Fetch data for a specific team.

### `/data_by_hitter/<hitter_id>`
- **Method:** GET
- **Description:** Fetch data for a specific hitter.

## Data Analysis and Modeling

The `data_model.py` script contains functions for analyzing the dataset and predicting performance metrics. The `womenintech_squared_up_rate_predict.ipynb` notebook provides detailed exploratory data analysis (EDA) and modeling.
